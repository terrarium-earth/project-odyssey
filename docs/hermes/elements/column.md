# `<column>` : Column

The `<column>` element is used in conjunction with the `<columns>` element to create a column in a row.

## Examples

The following will show 2 columns, each with 3 lines of text.

```html
<columns>
    <column>
        <p>Line 1</p>
        <p>Line 2</p>
        <p>Line 3</p>
    </column>
    <column>
        <p>Line 1</p>
        <p>Line 2</p>
        <p>Line 3</p>
    </column>
</columns>
```
