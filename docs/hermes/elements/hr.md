# `<hr>` : Horizontal Rule

The `<hr>` element is used to display a horizontal line.

## Attributes

[`color`](../attributes/color)
> The color of the line. Defaults to `gray`.

## Examples

The following example will display a red horizontal line.

```html
<hr color="red"/>
```
