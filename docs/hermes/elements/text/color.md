# `<color>` : Color Text

The `<color>` element is used to color the elements or text inside.

## Attributes

`color`
> A named or hexadecimal color value.

## Examples

The following will create a string with color text thats red.

```html
<text><color color="#ff0000">Text</color></text>
```
