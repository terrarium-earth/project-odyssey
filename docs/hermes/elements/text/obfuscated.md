# `<obfuscated>` : Obfuscated Text

The `<obfuscated>` element is used to obfuscate the elements or text inside.

## Alternatives

`<obf>`

## Examples

The following will create a string with obfuscated text.

```html
<text><obfuscated>Text</obfuscated></text>
```
