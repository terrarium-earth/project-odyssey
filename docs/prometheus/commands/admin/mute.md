# Mute Command

## Usage
---

### Muting a player

```text
/mute <player> [time]
```

Mutes the specified player for the specified amount of time. If no time is specified, the player will be muted for 50 years.

### Unmuting a player

```text
/unmute <player>
```

Unmutes the specified player.
