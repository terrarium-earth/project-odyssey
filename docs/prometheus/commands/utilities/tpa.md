# TPA Command

## Usage
---

### Teleporting to a player

```text
/tpa <player>
```

Requests to teleport to a player.

### Teleporting a player to you

```text
/tpahere <player>
```

Requests a player to teleport to you.

## Role Settings
---

:::info Default Options
- Expiration Time: 30 Seconds
:::
